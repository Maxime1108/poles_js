Structures itératives

Boucle FOR

Pour exécuter un certain nombre de fois une suite d'instructions, on utilise la boucle FOR. On va définir une variable qui va servir à compter le nombre de tout (=itération). On va définir une condition qui va permettre d'arrêter la boucle, quand cette condition sera fausse. On va définir un incrémentation (ou décrementation) de la variable qui sert à compter dles itérations.

Syntaxe :

for(initialisation ; condition ; incrémentation) {
... bloc d'instructions ...}

Boucle WHILE (tant que)

  • Syntaxe :

    while( condition ) {
    ...bloc d'instructions ...
}

  • Explication :

    Tant que la condition est vrai, on continue la boucle. ⚠ aux boucles infinies
