Portée des variables (= scope)

  • Une variable déclarée dasn une fonction n'est accessible que dans cette fonction. On dit que c'est une varaible locale à la fonction.
  • Une variable déclarée avec le mot clé var est une variable globale. Elle est accessible partout.
  • Une varaible déclarée avec le mot-clé let n'est accessible que dans le blocl où elle a été délcarée. Un block est délimité par des {} (accolades, curly barcket en anglais). On ne peut redéclarer une variable déclarée avec les let dans le même bloc.
  • Le mot const permet de déclarer des constantes. Une constante est une variable dont la valeur ne peut pas être modifiée. Comme pour let, const a une portée de bloc.
